Detailed Description

A Plot object is used to display data in a 2-D or 3-D plot.

\detail Using GnuPlot, this class is a C++ interface to plotting. for a list of GnuPlot commands, visit: http://www.duke.edu/~hpgavin/gnuplot.html or http://www.cs.uni.edu/Help/gnuplot/
